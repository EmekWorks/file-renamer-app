# File Renamer App

This project is a simple Python desktop app that renames multiple files in a selected folder.

The user chooses a folder, enters a new base name, and the program automatically renames all files in that folder.

## Features

- choose a folder
- count files in the selected folder
- rename multiple files at once
- save rename history to `rename_log.txt`
- simple desktop GUI built with tkinter
- standalone `.exe` version for Windows
